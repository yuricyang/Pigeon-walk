# Intro  
A snack game use gyroscope. User can rotate the phone to control the line direction and chase the red dot that randomly refreshed on screen. Also enable multiple users.<br>

# How to
1. Open the link with mobile phone. It also enable on the computer side but without the gyro function can only monitor the actions on the mobile.
2. Enable the gyro function. 
3. Tilt your phone 360 and chase the red dot refreshed.
4. Also enable for multiple players. Ask your friends to join!:)
